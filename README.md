## Getting Started

1. Install TypeScript globally:
   ```bash
   npm install -g typescript
   ```

2. Compile TypeScript files:
   ```bash
   tsc filename.ts
   ```

3. Run compiled JavaScript:
   ```bash
   node filename.js
   ```
